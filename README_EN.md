# FiveM React Boilerplate

[English Version](./README_EN.md) | [Português](./README.md)

This is a professional boilerplate for creating NUI interfaces in FiveM using **React**, **TypeScript**, and **Vite**.

## 🚀 How to use

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.

## 📚 Documentation

To understand how to use the included hooks, providers, and debug tools, access the detailed documentation:

- 🇧🇷 [Documentação em Português](./DOCUMENTATION.md)
- 🇺🇸 [Documentation in English](./DOCUMENTATION_EN.md)

## ✨ Features

- **Optimized NUI Communication**: Custom hooks for sending and receiving data.
- **Browser Development**: Debug tools to simulate the game without leaving the browser.
- **Visibility System**: Integrated control for interface display.
- **TypeScript**: Fully typed to prevent errors.
